# Använd global administration

Gäller företag.

Läs om fördelarna med Global Admin Console och se hur du får tillgång till dem. [Läs mer](#request-access-to-the-global-admin-console).

Skapa ett supportärende i Adobe Admin Console för att begära åtkomst till en Global Admin Console. Logga in på [Global Admin Console](https://adminconsole.adobe.com/support).

## Introduktion

[Global Admin Console](https://adminconsole.adobe.com/support) är till för kunder med komplexa organisationsstrukturer som antingen har många administrationskonsoler eller vill dela upp sina huvudkonsoler i flera konsoler. Exempel: multinationella företag, utbildningsföretag, storskolor och myndigheter. Den kapslar in befintliga Admin Consoles i en hierarkisk struktur, som ett organisationsschema, för att ge genomskinlighet i ett distribuerat företag.

Global Admin Console ersätter inte eller ändrar inte funktionaliteten för befintliga Admin Consoles. Det fungerar som en hierarki med en rotorganisation högst upp och alla konsoler som är kapslade under rotorganisationen. Det finns ett urvalskriterium som fokuserar på att köpa och skapa en hierarkistruktur där Admin Console fungerar som rotkonsol.

Mer information om Global Admin Console funktioner och fördelar finns i [rapporten om Global Admin Console](https://community.adobe.com/questions-624/new-white-paper-for-the-adobe-global-admin-console-678929#M35625) och i följande [video](https://youtu.be/FLBWR78wpok).

Innehållet och berättarrösten för den här videon finns för närvarande endast på engelska.

## Viktiga fördelar

Följande är några viktiga fördelar med [!DNL Global Admin Console]:

- Delegera administrationen av Adobe produktanvändning och molnlagring till divisioner, avdelningar och lokalkontor, vilket minskar behovet av centraliserad administration.
- Skapa organisationer för varje enhet och hantera dem i en hierarkisk struktur. Organisationer kan också tas bort.
- Distribuera resurser och produktlicenser mellan olika organisationer.
- Begränsa synligheten mellan organisationer så att administratörer endast kan visa användare och resurser inom sitt tilldelade omfång.
- Hantera Admin Console organisationsstruktur på ett självbetjäningssätt som passar organisationens katalogstruktur.
- Skapa administrationscentrum genom att utse administratörer som ska hantera Adobe-resurser för specifika organisationer eller projekt.
- Definiera och tillämpa policyer på organisationsnivå.
- Skapa, redigera och ta bort produktprofiler och användargrupper.
- Exportera organisationsinformation för rapportering eller användning i andra processer.
- Importera uppdateringar till organisationer och ändringar i organisationshierarkin.
- Få meddelanden om ETLA avtalsstatus i underordnade organisationer, inklusive kommande förfallodatum, utgångna eller inaktiva kontrakt. Mer information finns i dokumentationen för [kontraktets förfallodatum](https://helpx.adobe.com/enterprise/using/contract-expiry.html).

## Kriterier för deltagande

Adobe rekommenderar global administration om din organisation uppfyller ett eller flera av följande kriterier:

- Ni hanterar flera administrationskonsoler och vill ha central synlighet och kontroll.
- Du vill separera Admin Consoles och hantera dem oberoende av varandra.
- Du hanterar flera kontrakt.
- Du måste distribuera eller centralisera identitetskonfigurationen i alla underordnade organisationer.
- Du vill tilldela inköpta resurser till olika delar av företaget och tilldela separata administratörer.
- Du har administratörer för specifika Adobe-produkter, -regioner eller -organisationer och vill förhindra överlappning och -störningar.

## Begär åtkomst till Global Admin Console

Om du vill begära åtkomst till [Global Admin Console](https://adminconsole.adobe.com/support/) följer du stegen nedan efter att ha konsulterat din Adobe-representant.

1. Gå till [ > ](https://adminconsole.adobe.com/support/) i **[!UICONTROL Support]** Admin Console **[!UICONTROL Support Summary]**.

   >[!NOTE]
   >
   > Skicka supportärendet via huvudorganisationen, som fungerar som rotorganisation i hierarkin Global Admin.

1. Välj **[!UICONTROL Create Case]** för att öppna fönstret **[!UICONTROL Create Case]**.

1. Ange följande information i formuläret:

   - Ärendets prioritet: P4 - mindre
   - Effekter av problemet: Medium
   - Sammanfattning: &lt;Din organisations namn> begär åtkomst till Global Admin Console

1. I **[!UICONTROL Describe the nature of the issue you're facing]** anger du att din organisation begär åtkomst till Global Admin Console och inkluderar följande information:

   - **Namn och konsol-ID** (finns i URL:en som den serie siffror och bokstäver som föregår &quot;@AdobeOrg&quot;) för Admin Console som ska anges som *rotorganisation (överst)*, eller begär en ny konsol och ange önskat namn.
- I vissa fall rekommenderas en ny konsol. Adobe kan vägleda dig. Välj *root*-organisationen noggrant eftersom det är svårt att ändra senare.
   - Tilldela minst en aktuell systemadministratör till rollen Global administratör. Du kan lägga till fler administratörer senare.
   - Namn och e-postadress till din Adobe-representant eller Customer Success Manager.

   ![Ärendebeskrivning](/help/adobe-support-tools-guide/assets/describe.png)

1. Välj **[!UICONTROL Next]** och ange följande information:

   | Information som ska fyllas i | Beskrivning |
   | ------------------ | ----------- |
   | **Visa personer som du vill meddela** | Lägg till upp till tio e-postadresser för att få ärendeuppdateringar från Adobe kundtjänst. |
   | **Tidszon** | Adobe kundtjänst använder din tidszon om ett samtal krävs. Det här fältet fylls i automatiskt baserat på dina systeminställningar. |
   | **Arbetstid** | Adobe kundtjänst tar hänsyn till dina arbetstider och deras normala arbetstid om ett samtal behövs. Använd skjutreglaget för tid för att ange den bästa tiden att nå dig. |
   | **Telefonnummer** | Ange ett kontaktnummer så att Adobe kundtjänstmedarbetare kan kontakta dig för felsökning eller förtydliganden. |

1. Välj **[!UICONTROL Submit]**.
