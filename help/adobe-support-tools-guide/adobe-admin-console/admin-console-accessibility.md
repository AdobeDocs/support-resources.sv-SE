# Tillgänglighet i Adobe Admin Console

Gäller företag och team.

Lär dig mer om stöd för hjälpmedel i [!DNL Adobe Admin Console] som ger en central plats för att hantera dina Adobe-berättiganden i hela organisationen.

## Stöd för hjälpmedelsfunktioner i operativsystemet

Hjälpmedelsfunktionerna i många vanliga operativsystem kan användas på [!DNL Admin Console]. Dessa hjälpmedelsfunktioner inkluderar hög kontrast och andra visningsinställningar. Det kan dock finnas vissa begränsningar på grund av det aktuella användargränssnittet för [!DNL Admin Console].

## Stöd för tangentbord

Det mesta av det interaktiva innehållet i [!DNL Admin Console] kan navigeras och hanteras enbart med tangentbordet. Du kan till exempel använda **[!UICONTROL Tab]** eller **[!UICONTROL Shift]** + **[!UICONTROL Tab]** för navigering och kontroll. När du har navigerat till rätt länk eller knapp markerar du den genom att trycka på **[!UICONTROL Enter]** eller **[!UICONTROL Return]**. Det kan fortfarande finnas några undantag enligt beskrivningen i [Tillgänglighetskonformationsrapporten](https://www.adobe.com/accessibility/compliance.html) (_enbart på engelska_).

## Stöd för hjälpmedel

[!DNL Admin Console] har stöd för hjälpmedelstekniker (som omfattar skärmläsare och skärmförstorare). Mer information finns i [Rapport om överensstämmelse för tillgänglighet](https://www.adobe.com/accessibility/compliance.html).

## Vilka tillgänglighetsfunktioner stöds i Admin Console?

[[!DNL Admin Console]](https://adminconsole.adobe.com/) är en central plats för att hantera dina Adobe-berättiganden i hela organisationen. Den har stöd för hjälpmedelsteknik, tangentbordsstöd och hjälpmedel för operativsystem (med några undantag).

## Har du feedback?

[!DNL Admin Console]-teamet arbetar kontinuerligt för att förbättra sin tillgänglighet och användarupplevelse. Du kan dela med dig av dina synpunkter på tillgänglighetsfunktionerna i [Enterprise- och Teams-communityn](https://community.adobe.com/enterprise-teams-622).
